> Status: 執筆予定

## この章で学ぶこと

- 「繰り返す作業」を仕組みにする 3 つの拡張点と、その住み分け
- スラッシュコマンド: 定型の指示をコマンド化する（実例: /zenn-log）
- サブエージェント: 本体とは別の LLM に役割を分担させる（実例: MAGI の MELCHIOR / BALTHASAR / CASPER 多数決）
- フック（hooks): 特定の操作の前後に自動で処理を挟む（未着手なら「こう使うと効く」を読み物として）
- 「どれをいつ使うか」の判断軸
