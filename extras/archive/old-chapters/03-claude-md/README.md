## この章で学ぶこと

- CLAUDE.md とは「プロジェクトの記憶」であること

- 何を書くと効くか: 運用ルール・進捗・禁止事項・前提
- 「推測禁止」「過剰実装しない」のような指示の置き方
- 良い CLAUDE.md / 悪い CLAUDE.md の違い
- 実例として web-basics-handbook の CLAUDE.md を読む（推測禁止・3 箇所同期・身近な実例ルールなど）
