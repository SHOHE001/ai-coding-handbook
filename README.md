# Claude Code 実戦ハンドブック

非エンジニアの個人開発者が、**Claude Code を「完璧に使いこなせる」と言える状態**になるための実用ハンドブック。読み物ではなく、**自分で skill・コマンド・サブエージェント・フック・MCP を作って運用に乗せられる**ことを目標にする。

> 旧版は概念中心の読み物でしたが、実用・段階的（初級→上級）に全面作り替えました。旧原稿は `extras/archive/old-chapters/` に保全しています。

## このハンドブックの方針

- **実用メイン**: 各章は「仕組みは最小限 → 手順 → **コピペで動く最小例** → 実用パターン → つまずき」の順で進む
- **段階的**: 初級（土台・記憶・権限）→ 中級（コマンド/スキル・サブエージェント・フック）→ 上級（MCP・日常運用）
- **Windows 前提**: 実行環境は Windows 11 + PowerShell / Git Bash。例はこの環境で動く形で書く
- **正確性優先**: Claude Code の仕様は版で変わる。本書は確実に実在する核心を主軸にし、流動的な仕様は各章末の **公式ドキュメントリンク**で裏を取れるようにする
- **自作ツールを実例に**: MAGI・/zenn-log・/save・Notion/Slack/Gmail 連携など、実際に動かしているものを題材に「なぜこの形にしたか」を残す

## 学習ロードマップ（段階的）

| 段階 | 章 | 到達点 |
|---|---|---|
| **導入** | 第0章 | Claude Code の拡張点6つの地図が頭に入る |
| **初級** | 第1〜3章 | 基本操作・モード・CLAUDE.md・権限の安全な絞り方を押さえる |
| **中級** | 第4〜6章 | コマンド/スキル・サブエージェント・フックを自作して運用に乗せる |
| **上級** | 第7〜8章 | MCP で外部連携し、全部を日常の個人開発に組み込む |
| **発展・応用編** | 第10〜14章 | マルチエージェント・スキル育成・多層ガードレール・検証駆動・コスト戦略（組み合わせて実戦投入） |
| **発展・探究編** | 第9・15〜18章 | 自動化・プラグイン化・MCP自作・Agent SDK・演出（作る側／自動化を極める） |

> 発展コース（第9章以降）の各章は**独立した読み切り**。基礎コースのように順番に読む必要はなく、興味のある章から読めます。

## 読みかた

1. 章の `README.md` を読み、**コピペで動く最小例を実際に試す**
2. わからない単語が出てきたら遠慮なく Claude Code に聞く。回答と一緒に `extras/qa-log.md` に記録され、同じ関連章で3件たまると補強ページを Claude が提案してくれる
3. 各章末の **公式ドキュメントリンク**で最新仕様を確認する（版で変わるため）
4. 次の章へ
